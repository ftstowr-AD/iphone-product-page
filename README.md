Advance Project

## How to Open This Project in VS Code

1. Download or clone this repository.
2. Download and Open **Visual Studio Code**.
3. Click on **File → Open Folder** and select the project folder.
4. Open the `index.html` file.
5. Right-click and select **"Open with Live Server"** (if installed).

The HTML file will open in your browser, and the CSS will be applied automatically if it is linked properly.

### Note:

Make sure the CSS file is linked in your HTML like this:

```html
<link rel="stylesheet" href="style.css">
```
